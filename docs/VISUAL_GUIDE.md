## Color Legend

### Mode Colors
- **Easy Mode**: Green (#4CAF50)
- **Hard Mode**: Orange (#FF9800)
- **Practice Mode**: Purple (#9C27B0)

### Feedback Colors
- **Correct Answer**: Green background, green checkmark
- **Wrong Answer**: Red background, red X mark
- **Unselected**: Gray outline, white background
- **Selected (before answer)**: Teal/Primary color

### Theme
- **Primary**: Teal (#009688)
- **Background**: Gradient (Primary container â†’ Surface)
- **Text**: High contrast for readability
- **Cards**: Elevated with rounded corners (16px)
