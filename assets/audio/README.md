# Audio Assets Directory

This directory should contain Quran recitation audio files.

## File Naming Convention

Files should be named using the format: `{surahNumber}_{ayahNumber}.mp3`

### Examples:
- `1_1.mp3` - Surah Al-Fatihah, Ayah 1
- `2_1.mp3` - Surah Al-Baqarah, Ayah 1
- `2_255.mp3` - Surah Al-Baqarah, Ayah 255 (Ayat Al-Kursi)
- `36_1.mp3` - Surah Ya-Sin, Ayah 1

## Sources for Audio

You can obtain Quran recitation audio from:
- EveryAyah.com
- Quran.com
- Tanzil.net

## Recommended Reciters

- Sheikh Mishary Rashid Alafasy
- Sheikh Abdul Basit Abdul Samad
- Sheikh Saad Al-Ghamdi
- Sheikh Mahmoud Khalil Al-Hussary

## Setup

1. Download the audio files from your preferred source
2. Rename them according to the convention above
3. Place them in this directory
4. Optionally, upload to Cloudflare or another CDN for better performance

## Note

The app will work without audio files (demo mode), but audio playback features will show an error message when attempting to play.
